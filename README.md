# Suns Signal 001 - Collin Gillespie Contract Signal

This folder contains a reusable Phoenix Suns intelligence series format prepared by Accelerated Velocity Consulting.

Series name: `Suns Signal`

Naming pattern: `Suns Signal ### | [Featured Subject] Signal`

Edition label for this issue: `Father's Day Edition`

Audience note: this format is designed to feel personally curated for Mat Ishbia while keeping Accelerated Velocity Consulting discreet and behind the scenes.

Series structure:

- One featured Suns story per edition.
- Contract/roster/game context around the featured story.
- Public reaction from local media, national media, fans, X, Reddit, YouTube, podcasts, and blogs.
- Suns implications.
- Western Conference watch.
- Eastern Conference watch.
- League-level signal.

- `index.html` is the finished newsletter.
- `styles.css` controls the reusable AVC visual system.
- `sources.md` keeps the source ledger and method notes.
- `social-capture.md` keeps the Reddit/X/YouTube capture notes and confidence limitations.
- `assets/brand/logos/` contains the AVC logo assets used in the header/footer.
- `assets/contract-sentiment-cover.png` is the generated editorial cover visual for this edition.
- `assets/collin-gillespie-headshot.png` is the local player image used in the report.
- `assets/teams/suns/phoenix-suns-logo.png` is the local contextual Phoenix Suns team mark.
- `assets/og-collin-gillespie-contract.png` is the iMessage/Open Graph preview image.

For the next edition, duplicate this folder, increment the issue number, replace the featured story, hero copy, player or subject image, source ledger, reaction map, and data strip, then keep the same header/footer and section structure.

For iMessage previews, the page must be hosted at a public URL and the `og:image` metadata should point to the public URL for `assets/og-collin-gillespie-contract.png`.
